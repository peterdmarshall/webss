# webss

### Clone the repo
Navigate to your bin\
```cd /bin```\

Clone the repo\
```git clone git@github.com:peterdmarshall/webss.git```\

Add the folder to your path\
```export PATH=$PATH:/bin/webss```\

Make the script executable\
```cd webss```\
```chmod u+x webss```\
